# Keyboard Shortcuts

Speed up your Memos workflow with these keyboard shortcuts. Master these combinations to create, edit, and organize your memos more efficiently.

## Global Shortcuts

These shortcuts work throughout the Memos interface:

| Shortcut | Action | Description |
|----------|--------|-------------|
| <kbd>Ctrl</kbd>/<kbd>Cmd</kbd> + <kbd>N</kbd> | New memo | Create a new memo |
| <kbd>Ctrl</kbd>/<kbd>Cmd</kbd> + <kbd>K</kbd> | Search | Focus the search bar |
| <kbd>Ctrl</kbd>/<kbd>Cmd</kbd> + <kbd>/</kbd> | Command palette | Open command palette |
| <kbd>?</kbd> | Help | Show keyboard shortcuts help |
| <kbd>Esc</kbd> | Cancel/Close | Close modals or cancel actions |

### Browser Considerations

Some shortcuts may conflict with browser shortcuts:

- **Chrome/Edge**: <kbd>Ctrl</kbd>/<kbd>Cmd</kbd> + <kbd>K</kbd> conflicts with address bar
- **Firefox**: <kbd>Ctrl</kbd>/<kbd>Cmd</kbd> + <kbd>L</kbd> conflicts with location bar
- **Safari**: Various shortcuts may conflict with browser features

**Solution**: Use alternative shortcuts or disable browser shortcuts for Memos domain.

## Tips for Mastery

### Learning Strategy

1. **Start with basics** - Master global shortcuts first
2. **Practice daily** - Use shortcuts consistently for muscle memory
3. **Focus on your workflow** - Learn shortcuts for features you use most
4. **Gradual adoption** - Add 2-3 new shortcuts per week

### Productivity Tips

<Callout type="tip" title="Efficiency Boost">
**Power User Combo**: Use <kbd>Ctrl/Cmd + N</kbd> → type content → <kbd>Ctrl/Cmd + Enter</kbd> for lightning-fast memo creation!
</Callout>

- **Quick capture**: <kbd>Ctrl/Cmd + N</kbd> → write → <kbd>Ctrl/Cmd + Enter</kbd>
- **Batch editing**: Use multi-cursor for repetitive edits
- **Search everything**: <kbd>Ctrl/Cmd + K</kbd> is your best friend
- **Tag efficiently**: Type `#` and let autocomplete help

### Common Combinations

| Task | Shortcut Sequence | Description |
|------|------------------|-------------|
| Quick note | <kbd>Ctrl/Cmd + N</kbd> → type → <kbd>Ctrl/Cmd + Enter</kbd> | Fast memo creation |
| Find and edit | <kbd>Ctrl/Cmd + K</kbd> → search → <kbd>Enter</kbd> → <kbd>E</kbd> | Search and edit |
| Format text | Select text → <kbd>Ctrl/Cmd + B</kbd> → <kbd>Ctrl/Cmd + I</kbd> | Bold and italic |
