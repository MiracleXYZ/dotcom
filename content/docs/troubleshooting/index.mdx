# Troubleshooting

Find solutions to common issues when using Memos.

## Installation Issues

### Docker Container Won't Start

**Problem**: Container exits immediately or won't start.

**Solutions**:
1. Check container logs: `docker logs memos`
2. Verify port availability: `lsof -i :5230`
3. Check file permissions on data directory
4. Ensure sufficient disk space

### Permission Denied Errors

**Problem**: Database or file permission errors.

**Solutions**:
```bash
# Fix ownership of data directory
sudo chown -R 1001:1001 ~/.memos/

# Or use current user
sudo chown -R $(id -u):$(id -g) ~/.memos/
```
