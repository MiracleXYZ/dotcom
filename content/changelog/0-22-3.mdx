## Enhancements

- Tables now support rendering additional syntax;
- HTML elements rendering supported, now including `<br />`;
- Added a setting to enable link previews(_disabled by default_);
- Access token will be correctly deleted upon logout;
- Disable user registration has been moved to startup options/environment variables `--public`.
